# 更新日志

所有重要的项目变更都会记录在此文件中。

## [1.0.0] - 2024-10-11

### 新增

- ✨ 完整的用户认证系统
  - Google、GitHub、微信第三方登录
  - 登录状态持久化
  - 自动token刷新
  
- 📖 书签管理核心功能
  - 创建、编辑、删除页面
  - 添加、编辑、删除链接
  - 支持链接标签和子链接
  - 页面版本控制
  
- 🔗 权限分享系统
  - 生成只读链接
  - 生成编辑链接
  - 生成超级权限链接
  - 一键复制分享链接
  
- 🎨 极简UI设计
  - 类似X.com/ChatGPT的简约风格
  - 黑白灰主色调
  - 流畅的动画效果
  - 响应式布局
  
- 🌐 多环境支持
  - 自动切换API地址
  - 本地/测试/生产环境
  
- 💾 状态管理
  - Pinia状态管理
  - localStorage持久化
  - 路由守卫保护

### 技术栈

- Vue 3.4.21 (Composition API)
- Vite 5.1.5
- Pinia 2.1.7
- Vue Router 4.3.0
- Tailwind CSS 3.4.1
- Headless UI 1.7.17
- Axios 1.6.7

### 项目结构

- 清晰的目录组织
- 组件化开发
- 可复用的工具函数
- 统一的API封装
